정수 삼각형

백준 1932 문제 풀러가기

  • 난이도 : 실버1
  • 분류 : dp

삼각형 맨 위 꼭대기부터 시작하여 아래에 있는 수 중 하나를 선택하여 내려올 때, 선택된 수의 합이 최대가 되는 값을 구하는 문제 (단, 현재 층에서 대각선 왼쪽 아래, 대각선 오른쪽 아래만 선택하여 내려올 수 있다.)

총 3가지의 경우로 나누어 dp 테이블을 채워나가는 방식으로 풀이함.

  • 입력값과 동일한 dp 테이블을 생성, 삼각형의 맨 위 꼭짓점 값을 초기값으로 설정

  • 삼각형의 왼쪽 변에 위치한 값들은 대각선 오른쪽 위에서만 선택이 가능하므로 현재 값에 우상단 값을 더하여 dp table에 저장한다.

  • 삼각형의 오른쪽 변에 위치한 값들도 대각선 왼쪽 위에서만 선택이 가능하므로 현재 값에 좌상단 값을 더하여 dp table에 저장한다.

  • 가운데 위치한 값들은 대각선 오른쪽, 왼쪽 위에서 모두 선택이 가능하므로, 좌상단의 값과, 우상단의 값을 비교하여 그 중 더 큰 값을 더하여 dp table에 저장한다.

  • 합이 최대가 되는 경로의 합을 구하는 문제이기 때문에 dp table의 맨 마지막 값들 중 가장 큰 값을 return 한다.
